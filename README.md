# Dimas Website

Dimas Website adalah proyek website pribadi yang dibangun menggunakan **Next.js 15**, **TypeScript**, dan **Turbopack** untuk performa optimal. Website ini memiliki desain modern dan responsif menggunakan **Tailwind CSS**. Backend menggunakan **Laravel API** untuk mengelola konten secara dinamis. Manajemen package menggunakan **pnpm** untuk efisiensi dan kecepatan instalasi.
